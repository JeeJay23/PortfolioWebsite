Latchet & Morty

Een tool voor kinderen met aandachtsproblemen

Spelbeschrijving

Latchet & Morty is een game ontworpen om kinderen met concentratieproblemen te helpen door ze te trainen. De speler (het kind) loopt door een 3d-omgeving waarin hij meerdere opdrachten die te maken hebben met een bepaald vak moet voltooien. In het level zullen er meerdere afleidingen zijn die de speler moet weerstaan. Aan het einde van het level krijgt de speler een aantal punten gebaseerd op hoe snel de speler erover heeft gedaan. Het doel is om de speler hem af te leren om zich te laten afleiden.

Dit project was gemaakt met een team van zes personen, vijf daarvan zijn artists en een gamedeveloper. Het project was gemaakt in vier weken. 	public void AddBoxToSelectiondub(int val)
    {
        bool alreadyInList = false;

        for (int i = 0; i < b.assignment.boxesInput.Count; i++)
        {
            if (b.assignment.boxesInput[i] == val)
            {
                alreadyInList = true;
            }
        }

        if (!alreadyInList)
        {
            Debug.Log("value added");
            b.assignment.AddBoxToSelection(val);
        }
        else
        {
            Debug.Log("value removed");
            b.assignment.RemoveBoxFromSelection(val);
        }

        if (b.assignment.boxesInput.Count == b.requiredBoxes)
        {
            b.uiManager.contButton.SetActive(true);
        }

        b.uiManager.DrawBoxes();
    }
